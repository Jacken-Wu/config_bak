# How to Install joshuto

1. Clone the repo.

    ```bash
    git clone https://github.com/kamiyaa/joshuto.git
    cd joshuto
    git checkout v0.9.4
    ```

2. Use this command to install.

    ```bash
    sudo cargo install --path=. --force --root=/usr/local
    ```

3. Put keymap.toml into ~/.config/joshuto/.
