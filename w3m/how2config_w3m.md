# How to configure w3m

1. Install w3m and w3m-img

2. Open w3m, press `o`

    ```bash
    Display inline images - NO
    Use external image viewer - YES
    ```

3. Put `w3m-chafa-viewer` and `mailcap` intp `~/.w3m/`

4. Give execute permission to `w3m-chafa-viewer`

    ```bash
    chmod +x ~/.w3m/w3m-chafa-viewer
    ```
